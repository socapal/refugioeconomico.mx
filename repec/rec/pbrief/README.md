# Serie reservada: Policy Briefs

Esta carpeta existe para reservar la posible serie `RePEc:rec:pbrief`, pero no debe publicarse con metadata final hasta cerrar una regla institucional para el identificador de briefs.

Hoy el sitio usa `REC-PB-###` como codigo publico. Antes de crear `.rdf` reales para esta serie hace falta definir:

- si `Number` seguira una numeracion absoluta o anual;
- si la serie efectivamente vivira en RePEc;
- bajo que tipo oficial se publicaria, previsiblemente `ReDIF-Paper`;
- cual sera la URL canonica de la serie y de cada documento;
- que bloque adicional se agregaria a `repec/rec/recseri.rdf`.
